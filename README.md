# Análise de Vendas — Petshop Gatito (Power BI)

## Projeto: Análise de Vendas com Power BI

# Fase 1: Contexto do Problema

Uma rede fictícia de petshops deseja entender melhor o comportamento de vendas para tomar decisões estratégicas.
A empresa não possui visão clara sobre:

* Produto mais vendido
* Melhor região para abrir nova filial
* Evolução do faturamento ao longo do tempo
* Perfil de compra dos clientes

O objetivo deste projeto é transformar dados brutos em informações estratégicas através de um dashboard interativo no Power BI.

---

# Fase 2: Preparação dos Dados e Objetivo

O objetivo deste projeto é transformar dados brutos que estavam separados em múltiplos arquivos em informações estratégicas através de um dashboard interativo no Power BI.

Etapas realizadas:
* Limpeza de dados
* Padronização de datas
* Remoção de valores nulos
* Criação de chaves
* União das tabelas

# Insights encontrados

* Pet Shop fica localizado em Itaquera, justificando o dado histórico de faturamento na região. Em 2022 o bairro de Guaianases teve um grande volume de faturamento, potencial local para uma filial.
* No mês de dezembro o faturamento em 2022 caiu bruscamente.
* Maior parte do público do petshop é do gênero feminino.
* Maior volume de venda foram brinquedos para pet.
* Maior faturamento foi com produtos para cachorro.
